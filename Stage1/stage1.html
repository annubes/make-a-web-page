Lesson 1: The Basics of Web and HTML

Major Pieces of the Web

Computer Browser -----> The Internet ----> Servers

HTML ( Hyper Text Markup Language) is made out of:
  • Text content- what you see
  • Mark up- what it looks like
  • Reference to other documents
  • Links to other pages

HTML markup consists of an opening and closing tag and the whole thing is called an element. This video has a great explanation. Some tags do not need an end tag. The img and break tag do not need an end tag for example.There are also two types of container elements: Blocks which form an invisible box around the content and inline.

Tree Like Structure of HTML

HTML code resembles a tree. Each branch of the tree would resemble a box on the webpage. This feature makes HTML code easier to organize and represent on the webpage. When programmers talk about the DOM (Document Object Model) they are talking about the tree like structure of a page.

Lesson 3: Adding CSS style to HTML Structure

Cascading Style Sheets (CSS)

very useful tool for programmers to give style to an HTML page and avoide repetitions. For example if we want all h1 elements to have a black bacground and white text, we would give a code that would apply to all h1 elements on the page without the need for repetition.
